Documentation Page

The server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Display Welcome / GET None A text message welcoming user to application
Return a list of all movies /movies GET None A JSON object holding data about all movies
Return data concerning a specific movie by title /movies/[title] GET None A JSON object holding data concerning a specific movie (description, genre, director, actors, image URL, if it’s featured, release date, and movie rating)
Return a list of movies by genre /movies/genres/[genre] GET None A JSON object holding data about movies within a genre/s
Return data about a director by name /movies/directors/[name] GET None A JSON object holding data concerning a specific director (bio, birth year, death year)
New User Registration /users POST A JSON object holding data about the user to add A JSON object holding data about the user that was added
User info update (username, password, email, date of birth) /users/[username] PUT/PATCH None A text message indicating the user info that was updated
Add movie to list of favorites /users/[username]/favorites/[MovieID] POST A JSON object holding data about the movie to add A JSON object holding data about the movie added into the favorites list
Remove a movie from favorites list /users/[username]/favorites/[MovieID] DELETE None A text message stating if the movie had been successfully removed from favorites
User Account Deregistration /users/[username] DELETE None A text message indicating if the user had successfully deregistered
